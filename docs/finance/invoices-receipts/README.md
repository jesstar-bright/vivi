# Invoices & Receipts

Store or reference receipts for all business expenses here.

## How to Capture
1. **Email receipts:** Forward to a dedicated folder or email
2. **Screenshots:** Drop in this folder with date and vendor in filename
3. **Bank statements:** Monthly export from business account

## Naming Convention
```
YYYY-MM-DD_vendor_amount_category.pdf
```
Example: `2026-03-18_railway_5.00_infrastructure.pdf`

## Categories
- `infrastructure` — hosting, domains, storage
- `ai-ml` — Anthropic, Google AI, API costs
- `tools` — GitHub, Figma, analytics
- `app-store` — Apple/Google developer fees
- `marketing` — ads, influencer payments, content
- `education` — courses, books, conferences
- `equipment` — hardware, peripherals
- `legal` — LLC filing, contracts, trademarks
- `insurance` — business insurance
- `home-office` — proportional utilities, internet
