# Finance & Tax Documentation

Managed by the AI CFO agent. This is the source of truth for Crea's financial records.

```
finance/
├── r-and-d-credit/       ← R&D tax credit documentation (development logs)
├── llc-setup/            ← LLC formation, EIN, operating agreement, elections
├── expenses/             ← Monthly expense summaries (auto-generated where possible)
├── tax-calendar/         ← Quarterly deadlines, estimated payment tracking
├── invoices-receipts/    ← Receipt storage and categorization
└── api-costs/            ← Automated API usage/billing snapshots (Anthropic, Google, Railway)
```
