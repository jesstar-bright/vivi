# Ideas & Feature Backlog

Capture ideas here so they don't get lost. Each idea gets its own file with context.

Format for each idea file:
```
# [Feature Name]
**Date:** YYYY-MM-DD
**Priority:** high | medium | low | someday
**Effort:** small | medium | large

## The Problem
What pain point does this solve?

## The Idea
What would we build?

## Why Now / Why Later
Should we prioritize this or park it?

## Notes
Any context, screenshots, links, conversations that inspired this.
```
