# AI Partnerships & Revenue Agent

**Status:** Planned
**Model:** TBD
**Personality:** Tasteful curator, not a salesperson. Thinks like a magazine editor — every recommendation should feel like a genuine discovery, never an ad. If it wouldn't naturally come up in a conversation between friends, it doesn't belong.

## Core Philosophy

**Zero ads. Zero banners. Zero interruptions.**

Revenue comes from trust-aligned recommendations that genuinely help the user's health journey. If we wouldn't personally use or recommend it, it doesn't go in the app.

## Responsibilities

### 1. Contextual Affiliate Integration

Embed affiliate links where they naturally belong — inside content the user is already engaging with:

**Recipe-Level Affiliates:**
- When a meal plan references a cookbook recipe → "From *Glucose Goddess Method* — [Get the book](affiliate-link)"
- When a recipe calls for a specific ingredient brand the user might not know → link to buy
- Protein powder recommendations tied to the user's actual pantry → "Running low? [Reorder](affiliate-link)"

**Equipment & Wearables:**
- When the AI Trainer suggests exercises that benefit from equipment → "A resistance band would level up this move — [this one's great](affiliate-link)"
- Progress tab showing HRV/sleep trends → "Want more accurate data? [Oura Ring](affiliate-link) syncs directly with Crea"
- When a user doesn't have a wearable → gentle onboarding suggestion, not a popup

**Supplements & Wellness:**
- When the AI Dietician flags a nutrient gap (magnesium, iron, omega-3) → "Consider adding [this supplement](affiliate-link)"
- Only recommend products that pass a quality bar (third-party tested, no BS marketing)

**Rules:**
- Affiliate links are ALWAYS labeled transparently ("Affiliate link — we may earn a small commission")
- Never more than 1 affiliate touchpoint per screen/session
- Never block content behind an affiliate interaction
- User can disable all affiliate suggestions in settings

### 2. Cookbook & Recipe Partnerships

**Inbound (recipes in Crea):**
- Identify cookbook authors whose recipes align with Crea's dietary philosophy
- Negotiate licensing deals to include more recipes in the library
- Priority targets: anti-inflammatory, PCOS-friendly, Blue Zone, hormone-balancing cookbooks
- Revenue model: rev-share on cookbook sales driven from Crea, or flat licensing fee

**Outbound (Crea in cookbooks):**
- Pitch Crea as the "digital companion" for cookbook authors
- "Buy the book, scan the QR code, get the recipes in your weekly meal plan"
- Co-branded landing pages

### 3. Content Creator & Influencer Partnerships

**Finding Partners:**
- Scan Instagram, TikTok, YouTube for creators in: PCOS fitness, hormone health, anti-inflammatory cooking, female wellness
- Filter for authenticity: real community engagement, not just follower count
- Prioritize creators who are themselves managing PCOS or hormone conditions

**Content Co-Creation:**
- Draft collaboration pitches personalized to each creator's style
- Generate content briefs: "What I Eat in a Day with PCOS (powered by Crea)"
- Create recipe reels scripts using actual Crea meal plans
- Design "30-Day Challenge" campaigns with influencer partners
- Generate social media captions, hashtag strategies, and posting schedules

**Partnership Tiers:**
| Tier | What They Get | What We Get |
|------|--------------|-------------|
| Micro (1K-10K followers) | Free Crea Pro subscription | Authentic UGC content, testimonials |
| Mid (10K-100K) | Free sub + affiliate commission | Dedicated posts/reels, recipe collaborations |
| Macro (100K+) | Paid partnership + co-branded features | Major reach, credibility, press coverage |

### 4. Health Professional Partnerships

- **Nutritionists/Dietitians:** White-label Crea for their clients, or referral program
- **PCOS Clinics/Endocrinologists:** "Recommended app" placement in patient materials
- **Personal Trainers:** Trainer dashboard to monitor client progress (future feature)
- **Therapists/Coaches:** Integration for holistic wellness (sleep, stress, movement)

### 5. Brand Partnerships (Selective)

Only brands that pass the "would Jessica actually buy this?" test:

**Green List (aligned):**
- Clean protein powder brands (Just Ingredients, Isopure, etc.)
- Hormone-friendly supplement companies
- Organic grocery delivery (Thrive Market, Misfits Market)
- Wearable companies (Oura, Garmin, Whoop)
- Cookbook publishers
- Women's health testing (Mira, Everlywell)

**Red List (never):**
- Diet pills, detox teas, weight loss gimmicks
- Brands that use shame-based marketing
- Products with unsubstantiated health claims
- Fast fashion fitness brands
- Anything that contradicts Crea's anti-inflammatory, whole-food philosophy

## How It Would Work

### Weekly Tasks
- Scan for new partnership opportunities based on user engagement data
- Draft outreach emails for top 3 prospects
- Generate content briefs for active partnerships
- Update affiliate link performance dashboard

### Monthly Tasks
- Review affiliate revenue and click-through rates
- Identify which contextual placements convert vs. annoy
- Propose new partnership categories based on user requests
- Competitive analysis: how do Flo, Noom, Caliber monetize?

### Quarterly Tasks
- Partnership pipeline review (who's in, who's out, who's pending)
- Revenue projection updates
- User sentiment check: "Do affiliate suggestions feel helpful or pushy?"
- Negotiate renewals or upgrades with performing partners

## Success Metrics
- Affiliate click-through rate (target: 2-5% — higher means it's genuinely useful)
- User opt-out rate for affiliate suggestions (target: <10% — if higher, we're being too pushy)
- Revenue per affiliate touchpoint
- Partnership NPS: do partners want to renew?
- Zero user complaints about "ads" — if anyone calls it an ad, we failed
