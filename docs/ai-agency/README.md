# The AI Agency

Crea's team of AI agents — each with a defined role, personality, and set of responsibilities. Some are user-facing (dietician, trainer), others work behind the scenes (marketing, product, engineering).

## Architecture Principles
- Each agent has a **system prompt** that defines its expertise and personality
- Agents can **share context** (e.g., trainer tells dietician about workout intensity)
- User-facing agents should feel like **one cohesive team**, not disconnected bots
- Behind-the-scenes agents operate on **schedules or triggers**, not user requests
- All agent interactions are **logged** for quality review and improvement
